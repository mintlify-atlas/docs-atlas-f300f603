## System Requirements

Before installing Hydra, ensure your Mac meets these requirements:

### Hardware Requirements

- **CPU**: Intel or Apple Silicon processor
  - Apple Silicon (M1/M2/M3) recommended for best performance
  - Intel Macs supported but with lower performance
- **RAM**: 4GB minimum, 8GB or more recommended
- **GPU**: Metal-compatible graphics card (all Macs from 2012 onward)
- **Storage**: 
  - 500MB for the emulator
  - Additional space for games and firmware
  - SSD recommended for faster load times

### Software Requirements

- **Operating System**: macOS 10.15 (Catalina) or later
  - macOS 12 (Monterey) or later recommended
- **Xcode Command Line Tools** (for building from source)

## Installation Methods

<Tabs>
  <Tab title="Pre-built Release (Recommended)">
    
### Download Pre-built Binary

The easiest way to get started is to download a pre-built release.

<Steps>
  <Step title="Download Hydra">
    Visit the [latest release page](https://github.com/SamoZ256/hydra/releases/latest) and download the macOS app bundle.
  </Step>
  
  <Step title="Extract the Archive">
    Double-click the downloaded `.zip` or `.dmg` file to extract `Hydra.app`.
  </Step>
  
  <Step title="Move to Applications">
    Drag `Hydra.app` to your `/Applications` folder (optional but recommended).
  </Step>
  
  <Step title="First Launch">
    Right-click on `Hydra.app` and select **Open** to bypass macOS Gatekeeper if the app is not notarized.
    
    <Warning>
      On first launch, macOS may show a security warning. Click **Open** in the dialog to proceed.
    </Warning>
    
    Hydra will create its configuration directory at:
    ```
    ~/Library/Application Support/Hydra/
    ```
  </Step>
</Steps>

  </Tab>
  
  <Tab title="Build from Source">

### Build from Source

Building from source gives you access to the latest features and allows you to customize the build.

<Steps>
  <Step title="Install Dependencies">
    Install the required dependencies using Homebrew:
    
    ```bash
    brew install cmake ninja sdl3 fmt
    ```
    
    <Note>
      If you don't have Homebrew installed, get it from [brew.sh](https://brew.sh).
    </Note>
  </Step>
  
  <Step title="Clone the Repository">
    Clone the Hydra repository and initialize submodules:
    
    ```bash
    git clone https://github.com/SamoZ256/hydra.git
    cd hydra
    git submodule update --init --recursive
    ```
    
    <Warning>
      The `--recursive` flag is important - it ensures all required submodules are downloaded.
    </Warning>
  </Step>
  
  <Step title="Configure CMake">
    Configure the build with CMake:
    
    <CodeGroup>
    
    ```bash SDL3 Frontend (Default)
    cmake . -B build -G Ninja -DMACOS_BUNDLE=ON
    ```
    
    ```bash SwiftUI Frontend
    cmake . -B build -G Ninja -DMACOS_BUNDLE=ON -DFRONTEND=SwiftUI
    ```
    
    </CodeGroup>
    
    Available build options:
    - `MACOS_BUNDLE=ON` - Build a macOS app bundle
    - `FRONTEND=SwiftUI` - Use SwiftUI instead of SDL3
    - `HYPERVISOR_ENABLED=OFF` - Disable Apple Hypervisor backend
    - `CUBEB_ENABLED=OFF` - Disable Cubeb audio backend
  </Step>
  
  <Step title="Build with Ninja">
    Compile the project:
    
    ```bash
    ninja -C build
    ```
    
    This will take several minutes depending on your Mac's performance.
  </Step>
  
  <Step title="Locate the Built App">
    After building, you'll find the app at:
    
    ```
    build/bin/Hydra.app
    ```
    
    You can launch it directly or copy it to your Applications folder:
    
    ```bash
    cp -r build/bin/Hydra.app /Applications/
    ```
  </Step>
</Steps>

</Tabs>

## Firmware Setup

Many official Nintendo Switch games require system firmware files to run properly.

<Warning>
  You must dump firmware files from your own Nintendo Switch console. Hydra requires **decrypted** firmware files.
</Warning>

<Steps>
  <Step title="Obtain Firmware Files">
    Firmware files must be dumped from a Nintendo Switch console and decrypted. You can use tools like [switch-extract-macos](https://github.com/SamoZ256/switch-extract-macos) to decrypt firmware.
    
    <Note>
      Hydra does not include firmware files. You must provide your own legally obtained firmware.
    </Note>
  </Step>
  
  <Step title="Place Firmware Files">
    Copy your decrypted firmware files to a location on your Mac, such as:
    
    ```
    ~/Documents/Switch/Firmware/
    ```
  </Step>
  
  <Step title="Configure Firmware Path">
    Launch Hydra at least once to generate the configuration file, then edit:
    
    ```
    ~/Library/Application Support/Hydra/config.toml
    ```
    
    Set the firmware path:
    
    ```toml
    [System]
    firmware_path = "/Users/YOUR_USERNAME/Documents/Switch/Firmware"
    ```
    
    Replace `YOUR_USERNAME` with your actual macOS username.
  </Step>
</Steps>

<Steps>
  <Step title="Download Patches">

## Directory Structure

After installation and first launch, Hydra creates the following directories:

```
~/Library/Application Support/Hydra/
├── config.toml          # Main configuration file
├── sdmc/                # Virtual SD card storage
├── save/                # Game save data
└── sysmodules/          # System modules

~/Library/Logs/Hydra/    # Log files
```

## Verifying Installation

To verify Hydra is installed correctly:

<Steps>
  <Step title="Launch the Emulator">
    Double-click `Hydra.app` to launch the emulator.
  </Step>
  
  <Step title="Check Configuration File">
    Verify that the configuration file was created:
    
    ```bash
    ls ~/Library/Application\ Support/Hydra/config.toml
    ```
  </Step>
  
  <Step title="Review Settings">
    Open the config file to review default settings:
    
    ```bash
    cat ~/Library/Application\ Support/Hydra/config.toml
    ```
    
    You should see sections for CPU, Graphics, Audio, and System settings.
  </Step>
</Steps>

## Supported File Formats

Hydra supports the following Nintendo Switch file formats:

- **NRO** - Nintendo Relocatable Object (homebrew)
- **NCA** - Nintendo Content Archive (official games)
- **NX** - Alternative packaged format
- **NSO** - Nintendo Shared Object

<Note>
  **NSP files** are not directly supported. You can convert NSP to NCA using [switch-extract-macos](https://github.com/SamoZ256/switch-extract-macos) or to NX using [Herakles](https://github.com/SamoZ256/herakles).
</Note>

## Uninstallation

To completely remove Hydra from your Mac:

<Steps>
  <Step title="Delete the Application">
    ```bash
    rm -rf /Applications/Hydra.app
    ```
  </Step>
  
  <Step title="Remove User Data">
    ```bash
    rm -rf ~/Library/Application\ Support/Hydra
    rm -rf ~/Library/Logs/Hydra
    ```
    
    <Warning>
      This will delete all save data and configuration. Back up any important saves first.
    </Warning>
  </Step>
</Steps>
